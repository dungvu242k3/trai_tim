# Trái Tim Thủ Khoa Lý

Dự án hiệu ứng Visual "Trái tim thủ khoa Lý" - Phiên bản ổn định duy nhất.

## Tính năng
- Hiệu ứng hạt bụi hồng lãng mạn sử dụng HTML5 Canvas.
- Trái tim nhịp đập trung tâm.
- Tương thích tốt trên các trình duyệt hiện đại.

## Triển khai
Dự án được cấu hình để chạy trên Vercel.
